    \begin{indt}{\section{Opérateurs booléens}}
        $-$ Et : \texttt{\&\&} dans les deux langages

        $-$ Ou : \texttt{||} dans les deux langages

        $-$ Négation : \texttt{not} en OCaml, et \texttt{!} en C

        $-$ Test d'égalité : \texttt{=} en OCaml, et \texttt{==} en C. En OCaml, \texttt{==} teste l'égalité des adresses mémoires (à proscrire)

        $-$ Différence : \texttt{<>} en OCaml, et \texttt{!=} en C. En OCaml, \texttt{!=} teste la différence des adresses mémoire.
    \end{indt}

    \vspace{12pt}
    
    \begin{indt}{\section{Déclarations en OCaml}}
        $-$ Déclaration locale :
        \begin{lstlisting}[language=Caml, xleftmargin=40pt]
let nom = valeur in expression
(*Surtout pas : *)
let nom = valeur;
expression
        \end{lstlisting}

        $-$ Déclaration globale :
        \begin{lstlisting}[language=Caml, xleftmargin=40pt]
let nom = valeur
(*ou*)
let nom = valeur;;
        \end{lstlisting}
    \end{indt}

    \vspace{12pt}
    
    \begin{indt}{\section{Expressions conditionnelles}}
        \begin{lstlisting}[language=Caml, xleftmargin=40pt]
if condition then expression1 else expression2
        \end{lstlisting}

        avec \texttt{expression1} et \texttt{expression2} de \textbf{même type}.

        \begin{lstlisting}[language=Caml, xleftmargin=40pt]
x + (if b then 42 else 0)
        \end{lstlisting}

        Attention :
        \begin{lstlisting}[language=Caml, xleftmargin=40pt]
if b then
    Printf.printf "Message d'erreur\n";
    0
else
    expression
        \end{lstlisting}

        fait un \texttt{Syntax error} car le point virgule fait croire à OCaml que le \texttt{if} s'arrête à cet endroit.

        \vspace{12pt}
        
        \begin{lstlisting}[language=Caml, xleftmargin=40pt]
while b1 do
    if b2 then
        0
    else
        incr x
done;
!x
        \end{lstlisting}

        Plusieurs problèmes : pas le même type dans le \textit{if}. De plus, on ne peut pas arrêter la boucle \texttt{while} ainsi.

        On peut faire :
        \begin{lstlisting}[language=Caml, xleftmargin=40pt]
let b3 = ref true in
while b1 && !b3 do
    if b2 then begin
        b3 := false;
        x := 0
    end
    else
        incr x
done;
!x
        \end{lstlisting}

        ou :
        \begin{lstlisting}[language=Caml, xleftmargin=40pt]
exception Break

try
    while b1 do
        if b2 then
            raise Break
        else
            incr x
    done;
    !x
with
| Break -> 0
        \end{lstlisting}

        \vspace{12pt}
        
        \begin{lstlisting}[language=Caml, xleftmargin=40pt]
let l = [] in
for i = 0 to 42 do
    i::l
done;
l
        \end{lstlisting}

        Plusieurs erreurs. Version corrigée :
        \begin{lstlisting}[language=Caml, xleftmargin=40pt]
let l = ref [] in
for i = 0 to 43 do
    l := i::!l
done;
!l
        \end{lstlisting}

        \begin{indt}{Les seuls objets mutables sont :}
            $-$ Les cases des tableaux ;

            $-$ Les références ;

            $-$ Les champs d'enregistrement déclarés mutables.
        \end{indt}
    \end{indt}
